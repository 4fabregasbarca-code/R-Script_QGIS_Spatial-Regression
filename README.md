Data Input: Open Data on factor possibly influencing industry location
Geographical Data: Distance Matrix of German districts
Spatial Regression: OLS regression with spatial weights to account for spatial agglomoration
LISA: "LISA" map shows the Moran's I (spatial autocorrelation) -> high/low as significant autocorrelation
Main results: Rent, Technology, Skill, Access to Transportation, East/West were significant under spatial regression model
              Showcases knowledge spillovers and spatial agglomeration when industries decide where to locate
